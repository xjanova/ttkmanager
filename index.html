What it does today.

15 real shipping features across the foundation and pro tiers. The 16 advanced/integration features are scaffolded and unlock as the Marketing API is fully approved.

📊

Dashboard

KPI tiles for spend / CPM / CPC / ROAS, active rules, recent activity, and unread alerts in one glance.

🔑

Multi-Account OAuth

Connect any number of TikTok ad accounts. Out-of-band code paste; tokens encrypted at rest with DPAPI + AES-GCM.

⏰

Schedule Rules

Cron-based dayparting and budget control. 9 named presets (peak hours, weekdays, hourly testing) or any custom expression.

🤖

Auto-Rules Engine

If-then triggers on CPC, CPM, CTR, ROAS, spend, conversions, frequency. Pause losers, scale winners, alert on anomalies.

⚠️

Anomaly Detector

Z-score CPM and CPC against a 14-day baseline. Warning at 2σ, critical at 3σ. Alerts logged locally.

💰

Budget Pacing

Daily / weekly / monthly caps with optional auto-pause. Never blow the period budget.

📋

Bulk Operations

Multi-select campaigns and pause / enable / set-budget in one shot. Excel-flavored productivity.

📤

CSV Import / Export

Round-trip schedule rules through Excel. Export accounts and 10k-row audit log for offline analysis.

📈

Performance Reports

1-/7-/14-/30-day windows, per-campaign rollup with derived CPC, CPM, CTR, ROAS columns.

🔥

Dayparting Heatmap

7×24 hour-by-day grid colored by ROAS / CPM / CTR / Conversions. Find the hot windows.

📋

Activity Log

Every API call, scheduled action, retry, and error — searchable, exportable, 30-day default retention.

💾

Backup & Restore

Single .ttkbak archive with DB + settings. Auto-snapshot before destructive restores.

🩺

Health Check

SQLite, token store, Quartz, mock mode, disk — one panel, one click.

⚙️

Settings

Edit App ID, Secret, Redirect URI, DB path, mock mode. Persists to gitignored appsettings.local.json.

⌨️

Keyboard Shortcuts

Ctrl+1..8 to navigate, F5 to refresh, Ctrl+, for settings. Designed for power users.

Get running in 5 minutes.

.NET 9 SDK is the only requirement. The app is one .csproj — clone, build, run.

Step 1

Clone the repo

cd ttkmanager

Get the source from xman studio.

Step 2

Open in Visual Studio 2026 / Rider / VS Code

Double-click TTKManager.sln, then F5. Or run from CLI.

dotnet build TTKManager.sln
Step 3

Try the mock data

Default config runs in mock mode. Open Anomaly Detector → Seed demo samples to populate 14 days of synthetic metrics, then Scan Now to see real anomaly alerts.

Step 4

Connect your real account

Edit src/TTKManager.App/appsettings.local.json.example, save as appsettings.local.json, set your TikTokAppId, TikTokAppSecret, and flip UseMockApi: false. Restart the app.

Step 5

Build a portable .exe

./publish.cmd          # Windows
./publish.sh           # Bash / WSL

Output: publish/TTKManager.exe. Drop on a USB stick — no install, no .NET runtime required on target.

Step 6

Schedule your first rule

Connect an account → Schedules tab → pick a campaign → choose a cron preset like "Weekdays at 18:00" → set budget → Add Rule. The rule fires automatically.
