ยิงแอด TikTok บนออโตไพลอต. Run TikTok ads on autopilot.

โปรแกรมเดสก์ท็อป Windows แบบ portable ที่ตั้งเวลายิงงบ ปรับ pacing คุมเพดาน ยิง auto-rules ตาม metric และตรวจจับความผิดปกติของแคมเปญ — รองรับหลายบัญชีโฆษณา TikTok ในเครื่องเดียว ไม่มี backend ไม่มี telemetry token เข้ารหัสเก็บอยู่ในเครื่องคุณเอง A portable Windows desktop tool that schedules budgets, paces spend, fires auto-rules on metric thresholds, and detects performance anomalies — for one or many TikTok ad accounts. No backend, no telemetry, your tokens stay on your machine.

📊

Dashboard

การ์ด KPI ของ spend / CPM / CPC / ROAS, rule ที่ active, กิจกรรมล่าสุด และ alert ที่ยังไม่อ่าน — ดูได้ในหน้าเดียว KPI tiles for spend / CPM / CPC / ROAS, active rules, recent activity, and unread alerts in one glance.

🔑

Multi-Account OAuth

เชื่อมบัญชีโฆษณา TikTok ได้ไม่จำกัด ใช้ flow paste auth_code · token เข้ารหัสด้วย DPAPI + AES-GCM ในเครื่อง Connect any number of TikTok ad accounts. Out-of-band code paste; tokens encrypted at rest with DPAPI + AES-GCM.

⏰

Schedule Rules

ตั้งเวลาด้วย Cron + dayparting · 9 preset (peak hours, weekdays, hourly) หรือเขียน expression เอง Cron-based dayparting and budget control. 9 named presets (peak hours, weekdays, hourly testing) or any custom expression.

🤖

Auto-Rules Engine

If-then trigger ที่อิง CPC, CPM, CTR, ROAS, spend, conversions, frequency · pause ตัวที่แย่, scale ตัวที่ดี, alert เมื่อมี anomaly If-then triggers on CPC, CPM, CTR, ROAS, spend, conversions, frequency. Pause losers, scale winners, alert on anomalies.

⚠️

Anomaly Detector

คำนวณ z-score ของ CPM และ CPC เทียบ baseline 14 วัน · Warning ที่ 2σ, Critical ที่ 3σ · บันทึก alert ไว้ในเครื่อง Z-score CPM and CPC against a 14-day baseline. Warning at 2σ, critical at 3σ. Alerts logged locally.

💰

Budget Pacing

เพดาน Daily / Weekly / Monthly + auto-pause เมื่อใช้ครบ · ไม่มีทางเผลอใช้เกินงบ Daily / weekly / monthly caps with optional auto-pause. Never blow the period budget.

📋

Bulk Operations

เลือกแคมเปญหลายตัวพร้อมกัน → pause / enable / set-budget ทีเดียว — productivity เหมือน Excel Multi-select campaigns and pause / enable / set-budget in one shot. Excel-flavored productivity.

📤

CSV Import / Export

Round-trip ตาราง schedule rules ผ่าน Excel · export accounts และ audit log 10k แถวไปวิเคราะห์ offline Round-trip schedule rules through Excel. Export accounts and 10k-row audit log for offline analysis.

📈

Performance Reports

หน้าต่าง 1 / 7 / 14 / 30 วัน · สรุป per-campaign พร้อมคอลัมน์ CPC, CPM, CTR, ROAS 1-/7-/14-/30-day windows, per-campaign rollup with derived CPC, CPM, CTR, ROAS columns.

🔥

Dayparting Heatmap

ตาราง 7×24 ชั่วโมง×วัน เลือกระบายสีตาม ROAS / CPM / CTR / Conversions · หาช่วงทอง 7×24 hour-by-day grid colored by ROAS / CPM / CTR / Conversions. Find the hot windows.

📋

Activity Log

ทุก API call, scheduled action, retry, error · ค้นหาได้ · export ได้ · เก็บ default 30 วัน Every API call, scheduled action, retry, and error — searchable, exportable, 30-day default retention.

💾

Backup & Restore

ZIP ไฟล์เดียว .ttkbak (DB + settings) · auto-snapshot ก่อน restore เผื่อย้อนกลับ Single .ttkbak archive with DB + settings. Auto-snapshot before destructive restores.

🩺

Health Check

ตรวจ SQLite, token store, Quartz, mock mode, disk · พาเนลเดียว คลิกเดียว SQLite, token store, Quartz, mock mode, disk — one panel, one click.

⚙️

Settings

แก้ App ID / Secret / Redirect URI / DB path / Mock mode · เซฟลง appsettings.local.json Edit App ID, Secret, Redirect URI, DB path, mock mode. Persists to gitignored appsettings.local.json.

⌨️

Keyboard Shortcuts

Ctrl+1..8 สลับหน้า · F5 รีเฟรช · Ctrl+, เปิด Settings · ออกแบบสำหรับ power user Ctrl+1..8 to navigate, F5 to refresh, Ctrl+, for settings. Designed for power users.
